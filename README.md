# 🏧 ATM Simulator

A Python-based ATM simulation that includes both core banking logic and a graphical user interface (GUI).

## 🚀 Features
* **Account Management:** Simulates standard ATM operations like checking balances, depositing money, and withdrawing funds.
* **Graphical Interface:** Provides an interactive and user-friendly GUI for easy navigation.
* **Modular Design:** Clean separation between the visual interface and the backend transaction logic.

## 📋 Prerequisites
Ensure you have Python installed on your system. You can check your version by running:
`python --version` (Requires Python 3.x)

*(Note: Depending on the GUI library used, you may need to install additional dependencies like `tkinter` or `PyQt`, though `tkinter` is typically included with standard Python installations).*
